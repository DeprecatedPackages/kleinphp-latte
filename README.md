Klein.php with Latte template engine.

## Install

1) Install via composer.

```sh
composer create-project f3l1x/kleinlatte:~0.1
```

2) Create app/cache folder.

3) Open KleinPHP-Latte root folder and run **server.php**.

```php
php server.php
```
